# CUGCoding Forum

A lightweight graduation forum project rebuilt from scratch.

## Goals
- Simple and clear community forum
- Frontend-first development with Vue
- Core features only: auth, posts, comments

## Initial scope
- User login/register UI
- Post list and detail pages
- Comment posting
- Simple admin entry later

## Project structure
- `frontend/` Vue application
- `backend/` Spring Boot application
- `docs/` design notes and screenshots
